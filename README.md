## About The Project

Vocabify is a mobile-first web application designed to help users seamlessly learn, practice, and retain English vocabulary. 

Unlike standard dictionary apps, Vocabify is **specially built for learning word derivatives**. Understanding how a single base word transforms across different parts of speech is one of the most effective ways to achieve fluency and grammatical accuracy in English.

### Core Focus: Derivatives
The application organizes vocabulary by mapping out the complete family of a word:
**Noun → Verb → Adjective → Adverb**

**Example:**
* **Noun:** Beauty
* **Verb:** Beautify
* **Adjective:** Beautiful
* **Adverb:** Beautifully

By learning the complete derivative chain, users expand their working vocabulary exponentially rather than memorizing isolated words.

### Key Features
* **100% Offline Capable:** All vocabulary data and user progress are stored locally. No internet connection is required.
* **Native Pronunciation:** Built-in Text-to-Speech (TTS) utilizing British English voices to assist with accurate pronunciation of all word forms.
* **Material 3 UI/UX:** Smooth, fluid animations, haptic feedback, and a highly polished modern interface.
* **Theme Support:** Full system-aware dark and light mode support for comfortable reading in any environment.
* **Advanced Filtering:** Find exactly what you want to learn by filtering by Level, Part of Speech, Alphabet, or Theme.
* **Grammar Tips:** Quick, accessible grammar rules directly inside the application to help users identify noun, verb, adjective, and adverb suffixes.
* **Progress Tracking:** Set daily learning goals, mark words as learned, and track visual progress over time.
